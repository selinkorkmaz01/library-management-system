# Library Management System

A simple Java-based Library Management System developed using Object-Oriented Programming (OOP) principles.

## Features

- Add books
- Search books
- Display all books
- Book management using ArrayList

## Technologies Used

- Java
- Object-Oriented Programming (OOP)
- ArrayList

## Project Structure

- Book.java
- Library.java
- Main.java

## Purpose

This project was developed to practice object-oriented design, class relationships, and basic library record management.
